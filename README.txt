1.The GraphNet.py file contains the proposed network architecture code, which consists of three sub-networks: the preprocessing sub-network, the connection sub-network, and the graph convolution sub-network.
2.The conncetion_subnetwork.py file contains the code of the connection sub-network module
3.The graphconv_subnetwork.py file contains the code of the graph convolution sub-network module.
4.The graphattention_layer.py file contains the code of the graph attention layer in the graph convolution sub-network module.
5.The get_adj.py file contains the code of obtaining the adjacency matrix and degree matrix of the mesh nodes.
